本ファイルはAPKのビルド手順・配布手順を記載しない（`16` を参照する）。本ファイルが扱うのは「作成済みAPKを使って、UI・UXをどのように検証するか」である。

## 2. 検証の基本原則

1. 仕様書が完成しただけではPhase 15完了としない（実装・確認・再評価まで含めて完了）
2. Webだけの確認ではAndroid対応完了としない。Androidだけの確認ではWeb対応完了としない
3. 正常系だけでなく、空状態・絞り込み結果なし・エラー・処理中を確認する
4. 見た目だけでなく、操作・戻る・保存・コピー・再試行を確認する
5. UX-01〜UX-16の各項目に合格判定を持たせる（§14）
6. 検証中に別の問題を見つけても、その場で無制限に修正範囲を広げない（§18）
7. 不具合は再現条件と影響範囲を記録し、修正タスクを分ける
8. 各確認には §3 の判定ステータスを使用する
9. **実際に確認していない項目を推測で合格にしない**（未実施は必ず「未確認」とする）

## 3. 判定ステータス

| 判定 | 意味 |
|---|---|
| 未確認 | まだ確認を実施していない。推測で他の判定を付けない |
| 合格 | 確認を実施し、期待どおりであることを確認した |
| 条件付き合格 | 確認を実施し、下記の限定条件をすべて満たす軽微な問題のみが残る |
| 不合格 | 確認を実施し、期待を満たさない問題を確認した。修正タスクが必要 |
| 対象外 | この画面・環境ではその項目が該当しない（理由を記録する） |

**条件付き合格の限定条件**（すべて満たす場合のみ使用できる）。

* 機能は使用可能だが、軽微な視覚問題（微妙なズレ・余白の不揃い等）が残る
* 回避手段があり、今回の必須要件を阻害しない
* 残る問題が後続タスクへ正式に記録されている（`10-tasks.md` または `11-open-issues.md`）
* 安全性・保存・守秘・主要導線には影響しない

**条件付き合格にしてはいけないもの**（該当したら不合格とする）。

* DB保存の失敗・データ消失
* 画面が操作不能になる
* 守秘違反（守秘既定の破壊、守秘注意文の消失、現場情報の公開出力への混入）
* 戻れない（どの経路でもホームへ帰れない）
* クラッシュ

## 9. 状態別検証

| 状態 | 発生条件 | 期待表示 | 操作可能範囲 | 合格条件 |
|---|---|---|---|---|
| loading | データ取得中 | ActivityIndicator等の読み込み表示 | 取得対象以外の操作は可 | **emptyと同一表示にならない** |
| success | 取得成功・データあり | 一覧・内容の通常表示 | 全操作 | データが正しく表示される |
| empty | 取得成功・データ0件（条件なし） | 「データがない」旨＋作成への誘導 | 作成・戻る | loadingでもerrorでもないことが分かる |
| filtered-empty | 取得成功・絞り込み一致0件 | 「条件に一致しない」旨＋**条件解除方法の表示** | 条件変更・解除 | **emptyと区別できる** |
| error | 取得失敗 | 短い説明＋**再試行方法**＋データが消えたわけではない旨 | 再試行・戻る | **空状態として表示されない** |
| saving | 保存処理中 | ボタンdisabled＋処理中が分かる表示 | 保存以外（入力は不可でよい） | **二重操作を防止する** |
| copying | コピー処理中 | ボタンdisabled（長引く場合） | コピー以外 | 二重操作を防止する |
| copied | コピー成功 | 「コピーしました」等の文字表示 | 全操作 | 文字で結果が分かる。自動で消える場合も読み取れる時間（2秒程度以上）を確保する |
| copy-error | コピー失敗 | 「コピーできませんでした」等の文字表示 | 再コピー | 文字で結果が分かる。読み取れる時間を確保する |
| disabled | 入力不足・処理中等 | 非活性色＋押せない見た目 | その操作以外 | 押しても何も起きない。可能なら理由が近くに示される |

追加ルール。

* loading中にemptyを表示しない（取得完了までempty判定しない）
* errorを空状態として扱わない（catchで握りつぶさない）
* filtered-emptyには条件解除方法を表示する
* errorには再試行方法を表示する
* saving・copying中は二重操作を防止する
* success・errorの結果表示は、一定時間後に消える場合でも読み取れる時間を確保する
